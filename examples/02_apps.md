Apps describe the top level application

You can see this in the help:
```bash
$ 02_apps --help
MyApp 1.0

Kevin K. <kbknapp@gmail.com>

Does awesome things

USAGE:
    02_apps

OPTIONS:
    -h, --help       Print help information
    -V, --version    Print version information
```

And among the version
```bash
$ 02_apps --version
MyApp 1.0
```
