Subcommands can also have aliases

```
$ 21_aliases --help
MyApp 

USAGE:
    21_aliases[EXE] [SUBCOMMAND]

OPTIONS:
    -h, --help    Print help information

SUBCOMMANDS:
    help    Print this message or the help of the given subcommand(s)
    ls      Adds files to myapp
$ 21_aliases ls .
'myapp add' was used, input is: .
$ 21_aliases dir .
'myapp add' was used, input is: .
```
